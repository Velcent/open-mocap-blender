# open-mocap-blender
Open Mocap is an open-source Blender addon for AI-powered motion capture. It supports offline full-body pose tracking, real-time + offline hand tracking, and automatic retargeting to any rig. Designed for artists, animators, and researchers who want fast, accessible mocap inside Blenderâ€”no external software required.

Please note: The addon works only till Blender 4.0.0. Newer version of Blender doesn't support the addon.
